## NezaboodkaMySQL script to configure MySQL Server

```
Usage format:

	NezaboodkaMySQL.bat <action> <username> [<password>]

Where:
	<action> - one of the following:  
		/i  - install: create Nezaboodka Admin MySQL database   
		/r  - remove: drop all Nezaboodka MySQL databases (need confirmation)   
		/rq - remove quiet: drop all Nezaboodka MySQL databases without confirmation

	<username> - MySQL user account name
	<password> - MySQL user account password (optional)
```
